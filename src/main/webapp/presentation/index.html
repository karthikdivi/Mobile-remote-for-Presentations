What is real time web ???
-  The real-time web is a set of technologies and practices that enable users to receive information as soon as it is published by its authors, rather than requiring that they or their software check a source periodically for updates

What is it?

Bi-directional, Full Duplex and TCP for web
Saves 50% Server resources 
Proposed as part HTML 5
IETF Defined RFC 6455
W3C defined JavaScript API
Oracle defined JSR 356
Basic Idea
- Establish a single TCP connection
- Send Messages in both directions (Bi-Directional)
- Send messages independent of each other (Full Duplex)
-End the connection
